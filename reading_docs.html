Introduction to Linux/UNIX manpages

Manpages commands as well as listed stuff

To start off, the man command is very easy to invoke:

man ls

The above line will open the man page for ls in a terminal.

The apropos and whatis commands can take searches one step further, as these allow for a broader overview of your results.

Examples:

Search for descriptions that include all keywords

apropos -a git log

Print just the description of a man page for systemctl

whatis systemctl
